# 🎬 YouTube Transcript API

유튜브 영상의 자막 데이터를 추출하여  
텍스트 형태로 제공하는 간단한 API 서버입니다.

<br/>

# 🚀 Getting Started

```bash
pip install -r requirements.txt

uvicorn main:app --reload
```
